# Random-Website
A random website.

# For New Github users
Make sure Git is installed in your device

## Install Git
If not installed follow this steps:
If using windows:
run this command in the terminal to install git: winget install --id Git.Git -e --source winget
to make sure Git is successfully installed run this command in the terminal: git --version

# To contribute clone the repo
In your device run this command: git clone https://github.com/Ducktavian/Random-Website.git

Do your stuff..

Once done run these commands:
git add .
git commit -m "message of what feature or edit you did"
git push
